Trigonometric function system:$$\{\sin(0x), \cos(0x), \sin(1x), \cos(1x),\cdots, \sin(nx), \cos(nx)\}$$ Equivalent to: $$\{1, \sin(x), \cos(x),\cdots, \sin(nx), \cos(nx)\}$$ Proof the orthogonal of base: $$\int_{-\pi}^{\pi}\sin(nx)\cos(mx)dx, m\ne n$$ $$\int_{-\pi}^{\pi}\sin(nx)\sin(mx)dx, m\ne n$$ $$\int_{-\pi}^{\pi}\cos(nx)\cos(mx)dx, m\ne n$$ All you need is: $$\sin(nx)\cos(mx) = \frac{1}{2}\left(\sin(n+m)x + \sin(n-m)x\right)$$ $$\sin(nx)\sin(mx) = \frac{1}{2}\left(\cos(n-m)x - \cos(n+m)x\right)$$ $$\cos(nx)\cos(mx) = \frac{1}{2}\left(\cos(n+m)x + \cos(n-m)x\right)$$Another useful property: $$\forall{n}, \int_{-\pi}^{\pi}\sin^2(nx)dx = \int_{-\pi}^{\pi}\cos^2(nx)dx = \pi$$ There are many questions: 
	1. Why is the integral ?
	2. Why the limit is $[-\pi, \pi]$ ?
		Frist, we usually define **inner product** as $\int$ in **function space**, especially in $\mathcal{L}^2$ (If in complex space, we need to add a conjugate). Then, select this interval is for symmetry which is conventional to do **parity decomposition** (even-odd decompose) and in physical/realistic world, this interval definition is natural.



3. Fourier Series: 
	1. Cyclic Function:$$f(x) = f(x+2T), \text{here }T = 2\pi$$ 
	2. $$f(x) = \sum_{n=0}^{\infty}\left(a_n\sin(nx) + b_n\cos(nx)\right) $$ Or something like: $$f(x) = \frac{b_0}{2} + \sum_{n=1}^{\infty}\left(a_n\sin(nx)+b_n\cos(nx)\right)$$ 
	3. How to compute the coefficient ? 
		1. For $a_0$ we do this transformation: $$\int_{-\pi}^{\pi} f(x)dx$$ then the right of equation is equal to: $$\int_{-\pi}^{\pi}\frac{b_0}{2}dx + \int_{-\pi}^{\pi}\left(\sum_{n=1}^{\infty}(a_n\sin(nx)\right)dx + \int_{-\pi}^{\pi}\left(\sum_{n=1}^{\infty}b_n\cos(nx)\right)dx$$because of symmetry, this equal to: $$\int_{-\pi}^{\pi}\frac{b_0}{2}dx = \frac{b_0}{2}\int_{-\pi}^{\pi}dx = \left. \frac{b_0}{2}\right|_{-\pi}^{\pi} = b_0 \pi$$ then we reconstruct the equation to get $b_0$ :$$b_0 = \frac{1}{\pi}\int_{-\pi}^{\pi}f(x)dx$$
		2. For $a_n$ we do another transformation: $$\int_{-\pi}^{\pi}f(x)\sin(mx)dx$$according to the orthogonal of trigonometric , get right part: $$\int_{-\pi}^{\pi}\sum_{n=0}^{\infty}\left(a_n\sin(nx)\sin(mx) \right) dx+\int_{-\pi}^{\pi}\sum_{n=0}^{\infty}\left(b_n\cos(nx)\sin(mx) \right)dx$$ $$ = \int_{-\pi}^{\pi}\sum_{n=0}^{\infty}\left(a_n\sin(nx)\sin(mx)\right)dx = \int_{-\pi}^{\pi}a_n\sin^2(nx)dx = a_n\pi$$then $$a_n = \frac{1}{\pi}\int_{-\pi}^{\pi}f(x)\sin(nx)dx$$
		3. For $b_n$ , symmetry: $$\int_{-\pi}^{\pi}f(x)\cos(mx)dx$$then $$\int_{-\pi}^{\pi}\sum_{n=0}^{\infty}\left(a_n\sin(nx)\cos(mx) \right) dx+\int_{-\pi}^{\pi}\sum_{n=0}^{\infty}\left(b_n\cos(nx)\cos(mx) \right)dx = b_n\pi$$so $$b_n = \frac{1}{\pi}\int_{-\pi}^{\pi}f(x)\cos(nx)dx$$when $n=0$ this equation is equivalent to before derived $b_0$.
	4. For cycle is $2L$ instead of $2\pi$:
		1. $$f(t+2L) = f(t) \Rightarrow t = \frac{L}{\pi}x \Rightarrow f\left(\frac{L}{\pi}(x+2\pi)\right) $$
		2. Corresponding trigonometric and operator: $$\sin(nx) = \sin(\frac{n\pi}{L}t), \cos(nx) = \cos(\frac{n\pi}{L}t), \int_{-\pi}^{\pi}f(x)dx=\frac{\pi}{L}\int_{-L}^{L}f(t)dt$$so $$a_n=\frac{1}{\pi}\int_{-\pi}^{\pi}f(x)\sin(nx)dx = \frac{1}{L}\int_{-L}^{L}f(t)\sin\left(\frac{n\pi}{L}t\right)dt$$



4. Integrated with $\text{Eular Equation}$  $$ e^{i\theta} = \cos(\theta) + i\sin(\theta)$$First we add a angular velocity $\omega$ : $$\omega = \frac{2\pi}{T}, T=2L, x=\frac{\pi}{L}t \Rightarrow x = \omega t$$ $$f(x) = b_0 + \sum_{n=1}^{\infty}\left(a_n\sin(nx)+b_n\cos(nx)\right) \Rightarrow f(t) = b_0 + \sum_{n=1}^{\infty}\left(a_n\sin(n\omega t)+b_n\cos(n\omega t)\right)$$ Because the object is a cyclic function, so $$\forall{a, b}, |b-a|=T,\int_a^bf(x)dx = C$$ For the coefficients computation: $$a_n = \frac{1}{L}\int_{-L}^{L}f(t)\sin\left(\frac{n\pi}{L}t\right)dt = \frac{1}{L}\int_0^Tf(t)\sin(n\omega t)dt$$ $$b_n = \frac{1}{L}\int_{-L}^{L}f(t)\cos\left(\frac{n\pi}{L}t\right)dt = \frac{1}{L}\int_0^Tf(t)\cos(n\omega t)dt$$ $\text{Eular Equation}$ :$$\sin(\theta) = \frac{1}{2i}\left(e^{i\theta}-e^{-i\theta}\right),\cos(\theta) = \frac{1}{2}\left(e^{i\theta}+e^{-i\theta}\right)$$ Substitute $\sin(n\omega t)$ and $\cos(n\omega t)$we have: $$f(t) = \frac{b_0}{2} + \sum_{n=1}^{\infty}\left(\frac{a_n}{2i}\left(e^{in\omega t}-e^{-in\omega t}\right)+\frac{b_n}{2}\left(e^{in\omega t}+e^{-in\omega t}\right)\right)$$ $$ = \frac{b_0}{2} + \sum_{n=1}^{\infty}\left(e^{in\omega t}\frac{b_n-ia_n}{2}+e^{-in\omega t}\frac{b_n+ia_n}{2}\right)$$ $$=\frac{b_0}{2} +\sum_{n=1}^{\infty}\frac{b_n-ia_n}{2}e^{in\omega t} + \sum_{n=1}^{\infty}\frac{b_n+ia_n}{2}e^{-in\omega t} $$ $$ = \frac{b_0}{2} + \sum_{n=1}^{\infty}\frac{b_n-ia_n}{2}e^{in\omega t} + \sum_{n=-\infty}^{-1}\frac{b_{-n}+ia_{-n}}{2}e^{in\omega t}$$ $$= \sum_{-\infty}^{\infty}C_ne^{in\omega t}$$ $C_n$ is piecewise: $$C_n = 
		\begin{cases}
		\frac{b_{-n}+ia_{-n}}{2}, & \text{if } n \lt 0 \\
		b_0/2, &\text{if } n = 0 \\
		\frac{b_{n}-ia_{n}}{2}, & \text{if } n \gt 0 \\
		\end{cases}$$ we find: $$\frac{b_{-n} + ia_{-n}}{2} = \frac{1}{2L}\int_0^Tf(t)\left(-i\sin(n\omega t) + \cos\right(n\omega t))dt = \frac{1}{2L}\int_0^Tf(t)e^{-in\omega t}dt$$ $$\frac{b_n-ia_{n}}{2}=\frac{1}{2L}\int_0^Tf(t)\left(-i\sin(n\omega t) + \cos(n\omega t)\right)dt = \frac{1}{2L}\int_0^Tf(t)e^{-in\omega t}dt$$ $$\frac{b_0}{2} = \frac{1}{2L}\int_0^Tf(t)dt = \frac{1}{2L}\int_0^Tf(t)\left(i\sin(0)+\cos(0)\right)dt = \frac{1}{2L}\int_0^Tf(t)e^{-i0\omega t}dt$$ So indeed $C_n$ can be presented be a single paradigm:
		$$C_n = \frac{1}{2L}\int_0^Tf(t)e^{-in\omega t}dt$$

Fourier transform: $$f(t) = \sum_{-\infty}^{\infty}\frac{1}{2L}\int_{-L}^{L}f(t)e^{-in\omega_0 t}dt*e^{in\omega_0 t},T=2L, \omega_0\text{ is base frequency}$$ $$\Delta\omega_0 = (n+1)\omega_0 - (n)\omega_0 = \omega_0 = \frac{2\pi}{T}$$ $$ f(t) = \sum_{-\infty}^{\infty}\frac{1}{T}\int_{-L}^{L}f(t)e^{-in\omega_0 t}dt*e^{in\omega_0 t} = \sum_{-\infty}^{\infty}\frac{\Delta\omega}{2\pi}\int_{-L}^{L}f(t)e^{-in\omega_0 t}dt*e^{in\omega_0 t}$$ when $T\rightarrow \infty, \Delta\omega \rightarrow 0$: $$f(t) = \lim_{T\rightarrow \infty}\frac{1}{2\pi}\sum_{-\infty}^{\infty}\left(\int_{-\infty}^{\infty}f(t)e^{-in\omega_0 t}dt*e^{in\omega_0 t}\right)\Delta\omega = \frac{1}{2\pi}\int_{-\infty}^{\infty}\int_{-\infty}^{\infty}f(t)e^{-i\omega t}dte^{i\omega t}d\omega$$ So we got the FT and IFT: $$f(\omega) = \int_{-\infty}^{\infty}f(t)e^{-i\omega t}dt, f^{-1}\left(f(\omega)\right) = \frac{1}{2\pi}\int_{-\infty}^{\infty}f(\omega)d\omega$$
## Understand FT at Hilbert Space

A signal is a curve $f(x)$  in time domain, but in infinite space we can view it as a point. Such as point $\vec{x}$ : $$\vec{x} = (x_0, x_1, \cdots, x_n)$$ we can explain it as $x_i = f(i)$ so each point on the curve is just a project at the coordinate. We define the inner product as $$\langle u,v \rangle = \int_{-\infty}^{\infty} U(t)V(t)dt$$
Then it is a **Hilbert space**(A complete inner space). The system of trigonometric is just another **standard orthogonal base** in this space, but for convention we use **Eular equation** to present them. So we can do transition between them.  The coordinate's value is the **inner product** of original vector and new base: $$f(\omega) = \int_{-\infty}^{\infty}f(t)e^{-i\omega t}dt$$
