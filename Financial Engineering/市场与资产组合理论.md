## 有效市场假说
EMH： Effective Market **hypothesis**（有很多定义）
尤金：有效的信息可以立即无偏地在证券价格中得到反映，则认为市场是有效的
（人）闻风而动 + （信息）兔死狗烹
任何无风险套利消息都是转瞬即逝的

前提：
1. 理性预期理论：预期是基于当前所有信息和历史规律的最优认识
2. 市场只有无套利均衡
3. 信息已在价格中反应
4. 信息公开有效
5. 市场无摩擦

信息分类：1. 过去交易的信息，2. 可获得的公开信息 3. 所有可获得的信息
市场分类：1. 弱有效市场 2. 半强式有效市场 3. 强有效市场

因为不是所有人都是理性的，所以市场不是有效的！！！
市场只有可能是半强式有效，否则触发斯蒂格利茨悖论
## MM定理
收益与成本无关
融资活动不创造任何价值

1. 市场有效
2. 信息充分，不存在交易成本
3. 证券无限可分，投资者理性
4. 投资者具有预期一致性
5. 自由借贷，无违约风险

税盾效应导致修正MM定理，导致融资和收益有关（有税收问题）（但是没什么卵用，现实决定）

## 投资组合理论
1. 马科维茨投资组合理论：分散投资，复制资产，用均值-方差来衡量收益和风险
	+ 收益与风险匹配
	+ 投资者仅单期投资
	+ 无税
	+ 投资者理性（同风险最大收益，同收益最小风险）
	+ 预期收益和风险（取其收益的期望和方差）
	组合期望收益和成员期望收益呈线性，风险呈非线性
2. 有效组合边界
	可行集：投资者可能的投资组合
	 有效组合边界：全局最小方差以上的最小方差点
	 分离定理：有效边界上的点可以通过边界上的任意两点再组合得到
3. 资本市场线
	托宾： 无风险收益点出发和有效边界相切的射线 
	$$ E(r_Q) = wE(r_R) + (1-w)r_F, \sigma_Q = w\sigma_R$$
	$$ E(r_Q) = \frac{\sigma_Q}{\sigma_R}E(r_R) + \left(1-\frac{\sigma_Q}{\sigma_R}\right)r_F = r_F+\frac{E(r_R) - r_F}{\sigma_R}\sigma_Q$$	
4. 夏普比率
$$\frac{E(r_R) -r_F}{\sigma_R}$$
	就是夏普比率，表示单位风险的收益补偿

5. 分离定理
	切点之后的部分需要投资者依靠无风险利率借款进行投资
